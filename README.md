# Web Graphics & SVG Animation Showcase
This project is a collection of frontend experiments demonstrating HTML layout basics, CSS styling, and SVG animations. It is designed as a learning-focused showcase covering static page structure, visual galleries, and animated vector graphics.

Overview

The repository includes multiple HTML examples:

A simple photography-themed webpage layout using HTML and CSS

A comprehensive SVG animation demo covering transitions, transforms, and SMIL animations

An additional SVG-based Bubble Shooter game to demonstrate interactive SVG usage

Each section focuses on a different aspect of frontend development and visual rendering.

Contents
1. Photography Channel Layout

Navigation bar with logo and menu

Basic gallery structure for images

Float-based layout techniques

Clean and minimal styling for beginners

2. SVG Animations & Transitions Demo

CSS hover transitions on SVG elements

SVG motion animations using <animate>

Color transitions and opacity animations

Scale (pulse) and rotation animations

Path drawing animation using stroke properties

CSS keyframe animations applied to SVG shapes

These examples demonstrate both CSS-driven and SVG-native (SMIL) animation approaches.

3. SVG Bubble Shooter Game

An SVG-based Bubble Shooter game has also been added to demonstrate interactive gameplay mechanics using scalable vector graphics.

Technologies Used

HTML5

CSS3

SVG (Scalable Vector Graphics)

Vanilla JavaScript (where applicable)

How to Run

Download or clone the project files.

Open the respective .html files in a modern web browser.

All layouts and animations run directly without any setup.

No external libraries or frameworks are required.

Learning Goals

Understand page layout using HTML and CSS

Learn different SVG animation techniques

Compare CSS animations vs SVG SMIL animations

Explore interactive graphics using SVG

Notes

All examples are standalone and browser-based.

SVG animations may vary slightly depending on browser support.

This project is intended for educational and practice purposes.
